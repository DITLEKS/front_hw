# Чат с GigaChat

Это React-приложение для чата с ИИ GigaChat от Сбера.

## Функциональность

- Отправка сообщений пользователем
- Получение ответов от GigaChat
- Индикатор загрузки во время ожидания ответа
- Автоматическая прокрутка к последнему сообщению
- Блокировка интерфейса во время обработки

## Запуск проекта

### Предварительные требования

- Node.js (версия 18 или выше)
- npm

### Установка зависимостей

```bash
npm install
```

### Запуск прокси-сервера

Прокси-сервер необходим для обхода CORS-ограничений при работе с GigaChat API.

```bash
npm run server
```

Сервер запустится на порту 3001.

### Запуск React-приложения

В новом терминале:

```bash
npm start
```

Приложение запустится на порту 3000 (или 3002, если 3000 занят).

### Использование

1. Откройте браузер и перейдите по адресу http://localhost:3000 (или 3002)
2. Введите сообщение в поле ввода
3. Нажмите Enter или кнопку "Отправить"
4. Дождитесь ответа от GigaChat

```
src/
├── components/
│   ├── chat/
│   │   ├── ChatWindow.tsx    # Главный компонент чата
│   │   ├── MessageList.tsx   # Список сообщений
│   │   ├── Message.tsx       # Компонент сообщения
│   │   ├── InputArea.tsx     # Поле ввода
│   │   └── TypingIndicator.tsx # Индикатор печати
│   ├── layout/
│   │   └── AppLayout.tsx     # Макет приложения
│   └── settings/
│       └── SettingsPanel.tsx # Панель настроек
├── types/
│   ├── message.ts            # Типы сообщений
│   └── chat.ts               # Типы чата
└── mocks/
    ├── chats.ts              # Моковые данные чатов
    └── messages.ts           # Моковые сообщения
```

## API

Приложение использует GigaChat API через прокси-сервер:

- `/api/oauth` - получение access token
- `/api/chat` - отправка сообщений
