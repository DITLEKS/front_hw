# Чат с ИИ

Это React-приложение для чата с ИИ GigaChat.

## Функциональность

- Отправка сообщений пользователем
- Получение ответов от GigaChat
- Индикатор загрузки во время ожидания ответа
- Автоматическая прокрутка к последнему сообщению
- Блокировка интерфейса во время обработки

## Запуск проекта

### Предварительные требования

- Node.js (версия 18 или выше)
- npm
- GigaChat API токен (см. ниже)

### Получение GigaChat API токена

1. Перейдите на [личный кабинет GigaChat](https://developers.sber.ru/studio/adm-console)
2. В разделе "API ключи" создайте новый ключ
3. Скопируйте полученный токен (он выглядит как длинная строка в формате Base64)
4. Создайте файл `.env` в корне проекта:

```bash
cp .env.example .env
```

5. Откройте `.env` и замените `your_gigachat_token_here` на ваш токен:

```env
GIGACHAT_TOKEN=ваш_токен_здесь
```

### Установка зависимостей

```bash
npm install
```

### Запуск прокси-сервера

Прокси-сервер необходим для обхода CORS-ограничений при работе с GigaChat API.

```bash
npm run server
```

Сервер запустится на порту 3001.

### Запуск React-приложения

В новом терминале:

```bash
npm start
```

Приложение запустится на порту 3000 (или 3002, если 3000 занят).

### Использование

1. Откройте браузер и перейдите по адресу http://localhost:3000 (или 3002)
2. Выполните вход в приложение
3. Введите сообщение в поле ввода
4. Нажмите Enter или кнопку "Отправить"
5. Дождитесь ответа от GigaChat

## API

Приложение использует GigaChat API через прокси-сервер:

- `/api/oauth` - получение access token
- `/api/chat` - отправка сообщений
